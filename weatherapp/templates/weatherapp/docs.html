Technical Documentation for Web Application

1. App Objective

The primary objective of this web application is to provide users with interactive maps and real-time data visualizations. The platform combines features like weather mapping, geospatial analysis map of school and student location, and real-time flight tracking to offer diverse and useful geographic insights and the heat map showcasing the density of aircrafts. It also includes a home page for general navigation and an "About Us" page to present the application's background and purpose.

2. Technology Stack

  • Frontend:
    • HTML5, CSS3, JavaScript
    • Bootstrap
    • Leaflet.js
  • Backend:
    • Python Django
  • Database:
    • Postgres
    • PostGIS
  • APIs:
    • OpenWeatherMap API
    • OpenSkyNetwork API
  • Geospatial Server:
    • GeoServer

3. Functionality of Each System Component

  • Home Page:

    Provides a user-friendly interface for navigation to different sections of the app. Features an introduction to the app's purpose and key functionalities.

  • About Us Page:

    Details the mission, vision, and team behind the application. Includes contact information and links to external resources or social media.

  • Weather Maps:

    Displays real-time weather data using the OpenWeatherMap API. Allows users to input a location to view localized weather conditions.

  • Geospatial Analysis Maps:

    Interactive maps displaying the location of students and the schools of butwal area. Users can click the markers on the map to view informations of student and schools.

  • Real-Time Flight Map:

    Integrates the OpenSkyNetwork API to provide real-time flight tracking data. Allows users to view destinations and schedules.

  • Heat Map:

    Integrates the OpenSkyNetwork API to provide the real-time aircraft density. User can pan and zoom the map for further visualizaiton and understanding.

4. User Documentation

Step-by-Step Guide for Using the App:

  1. Access the Application: Open a web browser and navigate to the application’s URL.
  2. Home Page Navigation: Explore the introductory content and navigate to desired sections using the menu.
  3. About Us Page: Learn about the app’s background and team. Use the provided contact details for inquiries or feedback.
  4. Using the Weather Maps: Navigate to the Weather Maps section from the menu. Enter a location in the search bar. View real-time weather data, including temperature, wind speed, and humidity.
  5. Geospatial Analysis Map: Go to the Geospatial Map section. It displays the locaiton of each student and shows popup of the information of student when clicked. It also includes the hospitals of butwal area which also includes the popup feature showing the information of hospitals when clicked. Zoom and pan on the map for detailed exploration.
  6. Tracking Real-Time Flights: Access the Flight Map section. Search for specific area/Continent or view all flights within a selected region. Click on a flight to view detailed information such as destination and ETA.
  7. Heat Map: Visualize the aircraft density all over the world. User can pan and zoon to focus on specific areas.
  8. Contacting Support: Use the "Contact Us" section on the About Us page for assistance.
